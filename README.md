<!-- USAGE EXAMPLES -->
## Usage

To use the shell in interactive mode, type in the following and hit enter:
   ```sh
   ./hsh
   ```
For non-interactive mode you would call ./hsh.  An example:
   ```sh
   echo "/bin/ls" | ./hsh
   ```
To exit interactive mode:
   ```sh
   exit
   ```
